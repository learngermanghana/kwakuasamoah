# Kwaku Travel & Relocation Website

Next.js 15 + React 19 + TypeScript website scaffold for travel, relocation, and booking inquiries.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy to Vercel

1. Push this repository to GitHub.
2. Go to [vercel.com/new](https://vercel.com/new).
3. Import the repository.
4. Keep defaults (Framework: Next.js) and click **Deploy**.

Or with CLI:

```bash
npm i -g vercel
vercel
```

Follow prompts to link project and deploy.
